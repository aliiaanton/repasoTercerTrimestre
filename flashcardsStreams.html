Interfaz Funcional

Definición

Interfaz con un solo método abstracto (SAM). Se puede anotar con @FunctionalInterface (opcional pero recomendado).

Expresión Lambda

Definición y Sintaxis

Implementación concisa (en línea) de una interfaz funcional. Sintaxis: (parámetros) -> { cuerpo } o (parámetros) -> expresión.

Predicate

Propósito y Método

Evalúa una condición sobre un objeto T. Método: boolean test(T t). Usado comúnmente en filter().

Function

Propósito y Método

Transforma un objeto de tipo T en uno de tipo R. Método: R apply(T t). Usado comúnmente en map().

Consumer

Propósito y Método

Realiza una acción (efecto secundario) con un objeto T, no devuelve nada (void). Método: void accept(T t). Usado en forEach().

Referencia a Método

¿Qué es?

Forma aún más corta que una lambda para referirse a un método existente. Sintaxis: Clase::metodoEstatico, objeto::metodoInstancia, Clase::metodoInstancia, Clase::new.

Interfaz Stream

Definición

Secuencia de elementos que soporta operaciones de agregación (filter, map, reduce, etc.). No almacena elementos. Procesa datos de una fuente.

Operaciones Intermedias

Características y Ejemplos

Devuelven un nuevo Stream. Son "lazy" (perezosas, no se ejecutan hasta una op. terminal). Ej: filter(), map(), sorted(), distinct(), limit().

Tuberías (Pipelines)

Estructura

Cadena de operaciones de Stream: Fuente → 0+ Op. Intermedias → 1 Op. Terminal. Se evalúan de forma "lazy".

Operaciones Terminales

Características y Ejemplos

Consumen el Stream. Producen un resultado o efecto secundario. Son "eager" (ansiosas, disparan la ejecución). Ej: forEach(), collect(), reduce(), count(), findFirst().

filter()

Función

Selecciona elementos del Stream que cumplen una condición dada por un Predicate. Ej: stream.filter(n -> n > 10)

map()

Función

Transforma cada elemento del Stream aplicando una Function. Ej: stream.map(String::toUpperCase)

forEach()

Función

Realiza una acción (definida por un Consumer) para cada elemento del Stream. Ej: stream.forEach(System.out::println)

collect()

Función

Acumula los elementos del Stream en una estructura de datos (List, Set, Map) o un resumen. Usa la clase Collectors. Ej: stream.collect(Collectors.toList())
