Test de Repaso: JDBC

1. Las clases que componen la API de JDBC se encuentran localizadas en el paquete:





Respuesta correcta: c) java.sql

2. Las siglas URL se corresponden con:





Respuesta correcta: c) Uniform Resource Locator.

3. ¿Qué método de la clase Statement se utiliza principalmente para ejecutar sentencias DML (Data Manipulation Language) como INSERT, UPDATE o DELETE?





Respuesta correcta: c) executeUpdate() (Devuelve el número de filas afectadas).

4. SQL Injection es una técnica utilizada habitualmente para:





Respuesta correcta: b) Hackear aplicaciones que utilizan JDBC.

5. El formato de localizador de recurso que permite crear una conexión con un SGBD MySQL tiene el formato:





Respuesta correcta: a) jdbc:mysql:///

6. Al iterar sobre un ResultSet usando while(rs.next()), ¿cómo accedemos al valor de una columna llamada "nombre" de tipo String?





Respuesta correcta: c) rs.getString("nombre")

7. Al usar un PreparedStatement, ¿cómo se suelen representar los parámetros en la sentencia SQL antes de asignarles valor?





Respuesta correcta: c) Con el símbolo ? (signo de interrogación).

8. El acrónimo CRUD hace referencia a un conjunto de operaciones que incluyen:





Respuesta correcta: d) Eliminación, inserción, actualización y consulta de datos. (Delete, Create, Update, Read)

9. La técnica de mapeo objeto-relacional hace referencia a:





Respuesta correcta: c) Convertir cada atributo de un objeto en un campo de un registro en una base de datos relacional y viceversa.

10. ¿Cuál es el principal objetivo del patrón DAO (Data Access Object) en el contexto de JDBC?





Respuesta correcta: b) Encapsular la lógica de acceso a datos, separándola de la lógica de negocio.
