Test de Repaso: Colecciones Java

1. Collection es:





Respuesta correcta: a) Una interfaz

2. En el marco de trabajo Collection, una List es:





Respuesta correcta: c) Una colección cuya clase implementa la interfaz List

3. ¿Qué tipos permiten definir clases, interfaces y métodos capaces de funcionar con diversos tipos de datos?





Respuesta correcta: c) genéricos

4. Collections (con 's' final) es:





Respuesta correcta: d) Una clase con utilidades para el marco Collection

5. ¿Cuál de las siguientes es una implementación de List en la API de Java?





Respuesta correcta: b) ArrayList (LinkedList también lo es)

6. ¿Qué método se usa comúnmente para insertar un nuevo elemento en una colección?





Respuesta correcta: c) add()

7. Iterator es:





Respuesta correcta: b) Una interfaz

8. Los iteradores se usan principalmente para:





Respuesta correcta: c) Recorrer colecciones

9. La estructura Pila (Stack) se caracteriza porque:





Respuesta correcta: b) El primer elemento que entra es el último en salir (LIFO)

10. ¿Cuál de las siguientes es una implementación de Set en la API de Java?





Respuesta correcta: c) HashSet (TreeSet y LinkedHashSet también lo son)

11. En Java, Map es:





Respuesta correcta: a) Una interfaz

12. En un TreeMap, las entradas se van añadiendo por orden:





Respuesta correcta: c) natural de las claves (o según un Comparator)
