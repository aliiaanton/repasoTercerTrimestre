Tipos Genéricos (Generics)

¿Qué son y para qué sirven?

Mecanismo para parametrizar tipos (). En colecciones: especifican el tipo de contenido, dan seguridad en compilación y evitan casting.

Interfaz Collection

¿Qué es?

Interfaz raíz (excepto Maps). Representa un grupo de objetos. Métodos básicos: add(), remove(), size(), contains(), iterator().

Interfaz List

Características

Subinterfaz de Collection. Secuencia ordenada (mantiene orden inserción). Permite elementos duplicados. Acceso por índice (get(index)).

ArrayList

Descripción

Implementación de List (array dinámico). Rápido para acceso aleatorio (get/set). Lento para inserciones/eliminaciones en medio.

LinkedList

Descripción

Implementación de List (lista doblemente enlazada). Rápido para inserciones/eliminaciones (extremos). Lento para acceso aleatorio. Implementa Deque.

Interfaz Set

Característica Principal

Subinterfaz de Collection. Colección que NO permite elementos duplicados.

HashSet

Descripción

Implementación de Set (tabla hash). No garantiza orden. Más rápida (O(1) promedio). Permite un null.

TreeSet

Descripción

Implementación de Set (árbol). Elementos ordenados (natural o Comparator). O(log n). No permite nulls (por defecto).

LinkedHashSet

Descripción

Implementación de Set (hash + lista enlazada). Mantiene orden de inserción. Casi tan rápida como HashSet. Permite un null.

Conversiones List ↔ Set

¿Cómo?

Usando el constructor: new HashSet<>(list) para eliminar duplicados. new ArrayList<>(set) para obtener una lista.

Interfaz Map

¿Qué es?

Colección de pares clave-valor. Claves únicas. NO extiende Collection.

HashMap

Descripción

Implementación de Map (tabla hash). No garantiza orden. Más rápida (O(1) promedio). Permite clave null y valores null.

TreeMap

Descripción

Implementación de Map (árbol). Claves ordenadas (natural o Comparator). O(log n). No permite claves null.

LinkedHashMap

Descripción

Implementación de Map (hash + lista enlazada). Mantiene orden de inserción (o acceso). Casi tan rápida como HashMap. Permite clave null.

Vistas Collection de Map

¿Cuáles son?

Colecciones respaldadas por el Map: keySet() (Set de claves), values() (Collection de valores), entrySet() (Set de Map.Entry).

Clase Collections

¿Qué es?

Clase de utilidad (utility class) con métodos estáticos para operar sobre colecciones (sort, shuffle, reverse, max, etc.).
